## 说明

此资料包新增了原矿块的熔炉及高炉配方  
可将原矿块以9倍慢的方式烧成矿物块  
经验也会以9倍给予  

**支援的方块有：**
|              ID              | 中文名   |
|------------------------------|----------|
| `minecraft:raw_iron_block`   | 铁粗矿块 |
| `minecraft:raw_copper_block` | 铜粗矿块 |
| `minecraft:raw_gold_block`   | 金粗矿块 |
| `create:raw_zinc_block`      | 锌粗矿块 |
