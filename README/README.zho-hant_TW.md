## 說明

此資料包新增了原礦塊的熔爐及高爐配方  
可將原礦塊以9倍慢的方式燒成礦物塊  
經驗也會以9倍給予  

**支援的方塊有：**
|              ID              | 中文名   |
|------------------------------|----------|
| `minecraft:raw_iron_block`   | 鐵原礦塊 |
| `minecraft:raw_copper_block` | 銅原礦塊 |
| `minecraft:raw_gold_block`   | 金原礦塊 |
| `create:raw_zinc_block`      | 鋅原礦塊 |
